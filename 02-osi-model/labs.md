
Cisco Networking Basics Lab File – Suggested Labs

1. Lab: Device Types and OSI Layers
   
Objective: Identify network devices and map them to OSI layers.
Equipment/Tools: Cisco Packet Tracer or physical devices (PC, switch, router, hub).
Tasks:
Connect a PC → Switch → Router → Internet simulation.
Identify which OSI layer each device operates on:
Hub → Layer 1
Switch → Layer 2
Router → Layer 3
PC applications → Layer 7

Summary: Helps beginners connect theory (OSI) to real devices.
