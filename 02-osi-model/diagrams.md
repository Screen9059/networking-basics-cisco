
| Layer | Name             | Cisco/Networking Example             | Quick Tip / Function                           |
| ----- | ---------------- | ------------------------------------ | ---------------------------------------------- |
| 7     | **Application**  | Telnet, HTTP, SNMP                   | Interface for user apps to access network      |
| 6     | **Presentation** | SSL/TLS, Encryption                  | Translates, encrypts, or compresses data       |
| 5     | **Session**      | VPN connections, TCP sessions        | Manages sessions & connections between devices |

| 1     | **Physical**     | Cables, Hubs, NICs                   | Actual hardware, signals, and media            |
