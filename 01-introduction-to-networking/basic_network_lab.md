# Lab 1: Basic Network Setup (Router + Switch + Access Point)
  Cisco Packet Tracer

## Objective
Build a simple network using:
- Router
- Switch
- Access Point
- PCs and a Laptop

---

## Network Topology


---

## Devices Used
- 1 Router (e.g. 2911)
- 1 Switch (2960)
- 1 Access Point
- 2 PCs
- 1 Laptop

---

## Step 1: Add Devices
- Drag and drop all devices into the workspace.

---

## Step 2: Connect Devices
Use **Copper Straight-Through cables**:

- Router → Switch  
- Switch → PC1  
- Switch → PC2  
- Switch → Access Point  

Laptop connects via WiFi (no cable).

---

## Step 3: Configure Access Point
- Click Access Point  
- Go to Config  
- Set SSID: `MyNetwork`  

---

## Step 4: Connect Laptop to WiFi
- Open Laptop  
- Desktop → PC Wireless  
- Connect to `MyNetwork`

---

## Step 5: Verify Connections
- All cables should be green  
- Laptop should be connected to WiFi  

---

## What I Learned
- Difference between Router, Switch, and Access Point  
- Basic network topology  
- Wired vs Wireless connections
